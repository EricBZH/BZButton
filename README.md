中文介绍
==============
BZButton可以通过xib直接改变按钮跟文本的位置，并能直接在xib上直观看到效果。<br/>

安装
==============

### CocoaPods

1. 在 Podfile 中添加  `pod 'BZButton'`。
2. 执行 `pod install` 或 `pod update`。
3. 导入 "BZButton.h"。

```ruby
pod 'BZButton'
```

### 手动安装

1. 下载 BZButton 文件夹内的所有内容。
2. 将 BZButton 内的源文件添加(拖放)到你的工程。
3. 导入 "BZButton.h"。
